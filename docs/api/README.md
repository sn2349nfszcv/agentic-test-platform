# API Documentation

API documentation for Agentic Test Platform.

## Endpoints

- `GET /api/health` - Health check
- `GET /api/v1/agentic-test-platform` - List items
- `POST /api/v1/agentic-test-platform` - Create item
- `GET /api/v1/agentic-test-platform/:id` - Get item
- `PUT /api/v1/agentic-test-platform/:id` - Update item
- `DELETE /api/v1/agentic-test-platform/:id` - Delete item
